Manual del Usuario

Sistema VIS_TEC

Introducción

El presente manual tiene como finalidad guiar al usuario en el uso correcto del sistema de control de acceso, desarrollado para gestionar eficazmente el ingreso de visitantes y alumnos a la institución educativa, fortaleciendo los mecanismos de seguridad y generando reportes útiles para la toma de decisiones.

Este manual está dirigido a:

  • Usuarios visitantes que deseen registrarse para ingresar a la institución.
  • Estudiantes que requieran información institucional.
  • Administradores encargados de validar registros, generar reportes y supervisar la seguridad.

Requisitos del sistema

  • Dispositivo con acceso a internet (PC, tablet, smartphone).
  • Navegador actualizado (Chrome, Firefox, Edge).
  • Conexión a red institucional para los administradores.
  • Registro y autenticación de visitantes y alumnos.
  • Captura de datos de la visita.
  • Consulta de noticias (para alumnos).
  • Reportes de ingreso (solo administradores).

Acceso al sistema

Al acceder al sistema, se muestra la pantalla de inicio con opciones para:

  • Iniciar sesión como alumno
  • Iniciar sesión como visitante
  • Registrar nueva visita

1. Registro de Visitantes

Los visitantes deben llenar un formulario en línea con su información y el motivo de su visita. Completar el formulario con los siguientes campos:

  • Nombre
  • Apellidos
  • Teléfono (opcional)
  • Correo electrónico (obligatorio)
  • Asunto o motivo de la visita
  • Fecha programada de ingreso

Confirmar datos y hacer clic en “Enviar”.

2. Inicio de Sesión

Alumnos y visitantes pueden iniciar sesión con sus credenciales. El sistema distingue el tipo de usuario.

3. Agenda de Visitas

Los visitantes podrán consultar su cita. Los alumnos solo verán noticias relevantes.

4. Interfaz y Compatibilidad

Diseño responsive para facilitar el acceso desde móviles y computadoras.

Funciones del administrador

  • Validación de visitas: Confirmar que los datos ingresados son válidos.
  • Generación de reportes: Descargar registros de visitas por fecha, nombre o asunto.
  • Gestión de usuarios: Consultar y modificar información de visitantes o alumnos.

Seguridad del Sistema

  • Cifrado de contraseñas mediante hash seguro.
  • Validación de datos antes del almacenamiento.
  • Acceso restringido a datos sensibles solo al personal autorizado.
